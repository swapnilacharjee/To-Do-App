Welcome to Awesome ToDo! This is a modern, feature-rich task management application designed to help you stay organized and boost your productivity. Built with a focus on clean design, smooth user experience, and practical functionality.

Key Features

  • Create and manage tasks with subtasks
  • Mark tasks and subtasks as complete
  • Edit task titles on the fly
  • Password-protected delete functionality
  • Clear completed tasks with security
  • Real-time task statistics
  • Local storage for data persistence
  • Responsive design for all devices
